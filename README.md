# Inventory & Purchase Order Management System

A web-based system for managing inventory and purchase orders using Oracle
Database, PL/SQL, and Oracle APEX.

The system helps organizations track inventory levels, manage suppliers,
process purchase orders, and maintain a complete history of stock movements.

## Technologies
- Oracle Database
- PL/SQL
- Oracle APEX
- VS Code

## Key Features
- Inventory item management
- Supplier management
- Purchase order creation and approval
- Automated stock receiving and updates
- Inventory movement tracking
- Low stock monitoring

## System Workflow
1. Define inventory items and suppliers
2. Create and approve purchase orders
3. Receive items and update inventory
4. Monitor stock levels and movements
